# taxonomy

한국거래소 KIND 영업(잠정)실적 공정공시를 KRX Taxonomy로 XBRL/iXBRL화하고,
잠정↔확정(DART 정기보고서) 대사를 point-in-time으로 재현하는 **학습용 가상 KIND 뷰어**.

> 실제 한국거래소 KIND와 무관한 학습용 자료입니다.

## 뷰어 실행

`kind_project/viewer/index.html`을 브라우저로 열면 됩니다. 별도 서버·빌드 없이 동작합니다.

| 파일 | 역할 |
|---|---|
| `index.html` | 뷰어 마크업/스타일 |
| `app.js` | 목록·상세·잠정 신뢰도·XBRL 팩트 렌더링 |
| `data.js` | 공시 데이터(iXBRL 팩트·대사 결과 포함) |
| `plotly.min.js` | 차트 라이브러리 |

### 주요 기능
- **가정 오늘(as-of)**: 최상단에서 "오늘"로 가정할 날짜를 달력 또는 `YYYYMMDD` / `YYYY.MM.DD` / `YYYY-MM-DD`로 입력. 그 시점까지 접수된 공시만 노출.
- **시간흐름**: ×1 / ×43200 / ×86400 / ×172800 / ×345600 배속으로 시간을 흘려보내며, 그 사이 접수되는 공시가 목록에 실시간으로 나타남.
- **잠정 신뢰도**: 공시 시점(point-in-time) 기준으로 잠정 대비 확정 수정폭을 재현.
- **XBRL 팩트**: 태깅된 element·컨텍스트·단위를 마우스오버로 확인.
